Features

Roshin CMS enables developers to set up a quick headless backend for their text content to be managed and queried with a frontend UI framework.

GraphQL

  • Only get content you need
  • Efficient transfer of data
  • Manipulation of content

Content Management

  • Manage your content with a simple UI
  • Login Protected
  • Stored on a Database
